### 阶段一：转换为标准MCP格式（1天）

#### 1. 项目结构
```
bilibili-mcp-server/
├── src/
│   ├── index.ts          # 入口文件
│   ├── server.ts         # MCP服务器
│   ├── tools/            # 工具定义
│   │   ├── comment.ts    # 评论相关
│   │   ├── video.ts      # 视频相关

│   │   └── message.ts    # 消息相关

│   │   ├── bilibili.ts   # B站API
│   │   ├── wbi.ts        # WBI签名

├── README_zh.md
```

```typescript
// 从 Pi Agent 格式转换为 MCP 格式
```

| 风险 | 说明 | 应对 |
