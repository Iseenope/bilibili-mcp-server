## 🏛️ 平台背景

### 国家超算互联网是什么？

**国家级算力基础设施平台**，由中科院牵头，连接全国 20+ 超算智算中心：

- **定位**：打造国家先进算力底座，支撑"东数西算"工程
- **规模**：服务用户突破 80 万，应用商城商品超 7200 个
- **能力**：单日处理作业峰值突破 100 万个，月均访问量突破 1100 万
- **生态**：AI 社区开源大模型超 800 个，服务商超 500 家

### 运营方背景

**曙光智算** = 中科曙光（603019.SH）旗下子公司
- 中科院计算所背景
- 国内领先的超算/智算运营商
- 承担多个国家重点项目

## 🎯 活动详情

### 三类合作伙伴

| 伙伴类型 | 定位 | 任务 |
|----------|------|------|
| **智能体开发伙伴** | 开发者 | 发布智能体、MCP 服务、Skill、知识库 |
| **智能体体验伙伴** | 用户/体验官 | 使用智能体、提交反馈、提交使用案例 |
| **生态共创伙伴** | 企业/机构 | 1V1 专项对接，深度合作 |

### 可领取的资源

**大模型 API（各 500 万 Tokens）**：
- Qwen3-235B-A22B（通义千问旗舰模型）
- MiniMax-M2.5

### 任务类型

| 任务 | 说明 |
|------|------|
| 发布智能体 | 在平台创建并发布 AI 智能体 |
| 发布 MCP 服务 | 发布 Model Context Protocol 服务 |
| 发布 Skill | 发布技能插件 |
| 发布知识库 | 创建并发布知识库 |
| 智能体每日使用 | 日常使用智能体 |
| 提交有效反馈 | 提交产品反馈 |
| 提交真实使用案例 | 提交使用案例 |

### 奖励机制

- **Tokens 奖励**：即时到账
- **现金奖励**：按月结算，次月上旬发放
- **多层进阶收益**：商业加速、职业跃迁、巅峰席位等

## 🚀 建议行动方案

### 方案 A：轻量参与（推荐先试水）

**目标**：领取免费 API，体验平台

**步骤**：
1. 注册超算互联网账号
2. 填写申请表（选择"智能体体验伙伴"）

**收益**：1000 万 Tokens 免费额度

---

### 方案 B：深度参与（需要投入开发）

**目标**：发布智能体/MCP 服务，获取奖励

**步骤**：
1. 完成方案 A
2. 填写申请表（选择"智能体开发伙伴"）
3. 开发一个智能体或 MCP 服务
4. 发布到平台
5. 持续运营，获取使用量和奖励

**投入**：1-2 周开发时间
**收益**：Tokens + 现金奖励 + 平台曝光
